# Table service


### Survival
Does not send game end event
Does not send tournament winner event
Does not send money to bank



### Run service
```
make run
```
\* It will automatically launch mongodb in docker

### Testing
To run only e2e tests
```
make e2e_test
```
To run only unit tests
```
make unit_test
```
To run all tests
```
make test
```
